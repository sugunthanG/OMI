# 🟡 OMI - Orion Market Intelligence

A real-time AI-powered trading dashboard with live market signals, risk management, and WhatsApp alerts.

---

## 🚀 Features

- 📊 Live Market Data Visualization (Candlestick Chart)
- 🤖 AI-Based Trading Signals (BUY / SELL / NO TRADE)
- 📈 Confidence Score Prediction
- ⚡ Multi-Timeframe Analysis (5m / 15m / 1h)
- 🎯 Dynamic Stop Loss & Take Profit
- ⚙️ Risk Modes:
  - Conservative
  - Balanced
  - Aggressive
- 🔄 Auto Refresh System
- 🔊 Sound Alerts
- 📲 WhatsApp Alerts (Real-time)
- 📡 Trade Log Panel
- 🧠 Multi-Model Support (v2, v3, etc.)
